## 📝 ПРИМЕРЫ ИСПОЛЬЗОВАНИЯ:

### Простой чат:
```
1. Запустите ultimate-real
2. Нажмите [8] для quick setup
3. Используйте тестовый ключ (Y)
4. Нажмите [1] для чата
5. Введите: "Привет! Расскажи о себе"
```

### Смена модели:
```
1. Нажмите [2] в главном меню
2. Выберите категорию моделей
3. Введите номер модели
4. Модель автоматически переключится
```

### Настройка температуры:
```
1. Нажмите [4] для настроек
2. Выберите [1] для температуры
3. Введите значение (0.0-2.0)
   - 0.0 = детерминированный
   - 0.7 = сбалансированный
   - 1.5 = креативный
```

## 🐛 TROUBLESHOOTING:

### Если API не работает:
1. Проверьте интернет соединение
2. Проверьте правильность API ключа
3. Проверьте баланс на OpenRouter
4. Попробуйте другую модель

### Если модель не отвечает:
1. Некоторые модели могут быть перегружены
2. Попробуйте Gemini Flash или Claude Haiku
3. Уменьшите max_tokens

### Тестовый ключ не работает:
1. Возможно закончились кредиты
2. Создайте свой бесплатный аккаунт
3. Получите $1 бесплатно при регистрации
